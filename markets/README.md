# 金融市场路线图 (Financial Markets Roadmap)

> 从宏观视角理解各类金融市场的结构、参与者与交易机制。

## 学习目标

- 掌握金融市场的基本功能：资金配置、价格发现、风险分散、流动性提供。
- 区分主要市场类型：股票、债券、衍生品、外汇、大宗商品等。
- 理解主要市场参与者及其动机：个人投资者、机构投资者、做市商、监管机构。
- 了解市场运作流程：一级市场、二级市场、交易所 vs 场外 (OTC)。
- 结合市场数据和案例，加深对市场动态的理解。

## 学习顺序

1. **市场概览与核心功能**
   - 一级市场 vs 二级市场
   - 主要参与者与监管框架
   - 交易方式与基础术语
2. **股票市场 (Equity Markets)**
   - IPO、股票交易机制、指数
   - 市值结构、因子与风格
3. **固定收益/债券市场 (Fixed Income)**
   - 国债、公司债、收益率曲线
   - 信用评级、久期与凸性
4. **衍生品市场 (Derivatives)**
   - 期货、期权、互换 (Swaps)
   - 保证金、杠杆与对冲
5. **外汇与商品市场 (FX & Commodities)**
   - 汇率制度、主要货币对
   - 商品定价、供需驱动因素
6. **市场基础设施与监管**
   - 交易所、清算所、做市商角色
   - 风险事件与监管演变

## 学习方式建议

- 结合真实市场数据（Wind、Bloomberg、TradingView、FRED）观察价格和成交量变化。
- 跟踪各大交易所官网、监管机构公告，了解政策和基础设施。
- 阅读行业报告（投行研报、BIS、IMF）掌握市场结构变化。
- 将市场知识与会计、金融数学结合，分析资产定价逻辑。
