# 经济学基础路线图

> 面向零基础学习者，从“市场如何运作”入手，逐步建立经济学思维框架。

## 学习目标

- 搭建宏观与微观经济学的基本结构。
- 理解价格、供给、需求等核心概念以及它们如何决定市场结果。
- 初步掌握经济运行的关键指标与政策工具。
- 能够把经济学概念与日常生活、金融市场联系起来。

## 学习顺序

1. **市场基础认知**
   - 什么是经济学、稀缺性与选择
   - 机会成本与边际思维
   - 供给与需求的概念、图像和均衡
2. **消费与生产决策**
   - 弹性：价格、收入、交叉弹性
   - 消费者效用最大化的直观理解
   - 生产成本与规模经济
3. **市场结构**
   - 完全竞争、垄断、寡头的差异
   - 定价策略与市场效率
4. **宏观经济入门**
   - GDP、通胀、失业等核心指标
   - 经济周期与总需求 / 总供给
   - 财政政策与货币政策的基础逻辑
5. **开放经济与国际贸易**
   - 比较优势与贸易收益
   - 汇率、国际资本流动的直觉理解

> 按顺序完成每一节的笔记和练习，确保掌握关键术语与图表含义。

## 学习方式建议

- 每个主题配合阅读示例、绘制图表帮助理解。
- 使用真实案例（例如全球热门商品、央行政策）连接理论与现实。
- 看完一节就尝试用自己的话复述概念或解决小练习题。
