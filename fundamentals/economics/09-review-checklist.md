# 复习清单与自测（Review Checklist）

> 用作阶段总结：逐条核对已掌握的知识点，发现薄弱环节。

## 使用方式

- 在完成 `00`～`08` 章节后，利用本清单进行自我测验。
- 每项内容可打勾 ✅ 或写下疑问，方便回到对应章节复习。
- 建议配合纸笔演算、口头复述或与同伴讨论，强化记忆。

## 核心概念清单

- [ ] 能说明经济学研究的问题、解释稀缺性与机会成本。
- [ ] 会画供需曲线，判断均衡、短缺、过剩以及非价格因素导致的曲线移动。
- [ ] 理解弹性的定义、中点法计算以及弹性对总收入的影响。
- [ ] 能够区分消费者剩余和生产者剩余，运用面积计算交易福利变化。
- [ ] 熟悉完全竞争、垄断竞争、寡头、垄断的区别，并能举现实例子。
- [ ] 掌握 GDP 的构成、名义与实际的区别，并能解释 GDP 平减指数。
- [ ] 能解读 CPI、核心 CPI、失业率、劳动参与率等指标，理解菲利普斯曲线。
- [ ] 了解财政与货币政策工具的作用与局限，理解政策协同的意义。
- [ ] 理解比较优势、贸易收益与关税/配额的影响。

## 练习建议

- 选择任一宏观事件（如疫情、能源危机），尝试用供需、弹性、政策工具分析其影响。
- 从国家统计局或世界银行下载最新 GDP、CPI、失业率数据，撰写简短解读。
- 对某个行业进行市场结构分析，判断其竞争形态与可能的政策关注点。
