# 时间价值入门 (Time Value of Money Basics)

> 今天的一元钱和明天的一元钱不一样：时间会改变货币价值。

## 学习目标

- 理解货币时间价值的核心逻辑：资金具有成本与机会成本。
- 掌握现值 (PV)、终值 (FV)、利率 (r)、期数 (n) 的基本概念。
- 学会简单的现值、终值计算，并理解贴现的意义。
- 明白复利与单利的区别。

## 为什么会有时间价值？

- **机会成本**：资金今天就可以投资、消费，推迟使用意味着放弃收益。
- **风险因素**：未来现金流存在不确定性，需要折现考虑风险补偿。
- **通货膨胀**：货币购买力随时间降低，需要更高金额才能买同样东西。
- **偏好因素**：人们通常偏好尽早获取收益、推迟支出。

## 基本符号与关系

- 现值 (Present Value, PV)：今天的价值。
- 终值 (Future Value, FV)：在未来某一时点的价值。
- 利率 (Interest Rate, r)：每期的回报率或贴现率。
- 期数 (Number of Periods, n)：复利或贴现的期数。

简单关系：

```math
FV = PV \, (1 + r)^n \qquad \text{或} \qquad PV = \frac{FV}{(1 + r)^n}
```

例：今天存入 10,000 元，年利率 5%，一年后价值多少？

```
FV = 10,000 × (1 + 0.05)^1 = 10,500
```

如果希望一年后得到 10,500 元，贴现率 5%，现值是多少？

```
PV = 10,500 / (1 + 0.05) = 10,000
```

## 单利 vs. 复利

- **单利 (Simple Interest)**：只对本金计息。
  ```math
  FV = PV \,(1 + r \times n)
  ```
- **复利 (Compound Interest)**：利息也会产生利息。
  ```math
  FV = PV \,(1 + r)^n
  ```

复利增长更快，期数越多差异越显著。

例：本金 10,000，年利率 5%，三年后：

- 单利：`FV = 10,000 × (1 + 0.05 × 3) = 11,500`
- 复利：`FV = 10,000 × (1 + 0.05)^3 ≈ 11,576.25`

## 贴现的含义

- 未来现金流折算到今天的价值，即 PV。
- 贴现率越高，现值越低（承担的机会成本、风险越大）。
- 贴现是投资决策的核心：比较项目的 PV 与初始成本，判断是否值得投入。

## 快速自测

1. 年利率 6%，复利计息，三年后要达到 12,000 元，今天需要存入多少？
2. 单利和复利的差异有什么实际意义？列举一个场景说明选择复利更合理。
3. 如果贴现率从 5% 提升到 8%，同一笔未来 10,000 元的现值如何变化？

### 参考答案与解析

1. `PV = 12,000 / (1 + 0.06)^3 ≈ 10,079.37`
2. 单利忽略利滚利，适合短期或固定利息协议；复利考虑利滚利，适合银行存款、长期投资。复利能更准确反映资金时间价值。
3. 贴现率越高，现值越小。例如：`PV_5% = 10,000 / (1.05) ≈ 9,523.81`；`PV_8% ≈ 9,259.26`。
