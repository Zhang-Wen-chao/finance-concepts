# 金融数学路线图

> 以时间价值为核心，循序渐进掌握金融计算的基本工具。

## 学习目标

- 理解货币时间价值概念，熟练进行现值、终值、折现率等计算。
- 掌握年金、永续年金、等额本息等常见现金流模型。
- 熟悉利率换算、连续复利、贴现因子的使用。
- 为后续的投资评估、债券定价、量化分析打好数学基础。

## 学习顺序

1. **基础概念与时间价值**
   - 会计利率 vs. 实际利率、复利频率
   - 现值、终值、贴现因子
   - 单笔金额的时间价值计算
2. **年金与系列现金流**
   - 普通年金、预付年金
   - 永续年金
   - 等额本息贷款
3. **利率与收益率换算**
   - 名义利率、实际利率、年化收益
   - 复利、连续复利、对数收益率
4. **贴现曲线与现金流分析**
   - 贴现因子、零息利率
   - 净现值 (NPV)、内部收益率 (IRR)
   - 应用：债券定价、项目投资评估

> 建议配合编程工具（Excel、Python）完成计算练习，加深理解。

## 学习方式建议

- 每个概念都写出公式，并尝试手算例题。
- 使用 Excel 的 `FV`, `PV`, `RATE`, `NPER` 等函数进行验证。
- 结合真实案例（如银行贷款、定期存款、债券收益）练习。
- 对于连续复利、收益率换算，可在 Python 中编写简短脚本计算。
