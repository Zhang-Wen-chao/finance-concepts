# 会计等式与借贷记账

> 会计的语言从一条简单的等式开始：资产 = 负债 + 所有者权益。

## 学习目标

- 理解资产、负债、所有者权益的含义与分类。
- 掌握会计等式如何保持恒等，以及交易对等式的影响。
- 学会借贷记账法的基本规则，厘清“借”“贷”并非现实中的借款/贷款含义。
- 能够用 T 字账户记录简单的经济业务。

## 基本概念

- **资产 (Assets)**：企业拥有或控制的、能带来经济利益的资源（现金、存货、设备、应收账款等）。
- **负债 (Liabilities)**：企业承担的、需要偿还的债务或义务（应付账款、借款、应付工资等）。
- **所有者权益 (Equity)**：资产扣除负债后的剩余权益，反映股东在企业中的剩余利益（实收资本、资本公积、留存收益）。

会计等式保持平衡：  
`资产 = 负债 + 所有者权益`

任何交易都必须让等式左右两侧保持相等。

## 交易示例：如何影响等式

| 交易 | 资产 | 负债 | 所有者权益 |
| --- | --- | --- | --- |
| 股东投入 10 万现金 | +10 万 | 0 | +10 万 |
| 银行借款 5 万到公司账户 | +5 万 | +5 万 | 0 |
| 用现金购置设备 8 万 | -8 万 | 0 | 0 |
| 赊购存货 2 万 | +2 万 | +2 万 | 0 |
| 销售产品收款 3 万（成本 1.5 万） | +3 万 | 0 | +1.5 万（毛利） |
| 支付工资 1 万 | -1 万 | 0 | -1 万 |

每笔交易都让左右两侧同时变动，保持等式成立。

## 借贷记账法基础

借贷记账法是双重记账制度的核心，每个交易同时记录至少两个项目：借方与贷方。

### 借贷规则（简记法）

| 会计要素 | 借方 | 贷方 |
| --- | --- | --- |
| 资产 | 增加 | 减少 |
| 负债 | 减少 | 增加 |
| 所有者权益 | 减少 | 增加 |
| 收入 | 减少 | 增加 |
| 费用 | 增加 | 减少 |

> “借”“贷”只表示方向，与银行中“借钱/贷款”无直接关系。

### T 字账户

```
       现金
借方             贷方
----------------------
100,000（投入）   8,000（购设备）
 5,000（借款）    1,000（付工资）
 3,000（销售收款）
```

借方记录现金的增加，贷方记录现金的减少，期末余额在借方。

## 常见业务分录示例

1. **股东出资**：借 `现金` 100,000；贷 `实收资本` 100,000
2. **银行贷款**：借 `现金` 50,000；贷 `短期借款` 50,000
3. **购买设备（现金支付）**：借 `固定资产` 80,000；贷 `现金` 80,000
4. **赊购原材料**：借 `存货` 20,000；贷 `应付账款` 20,000
5. **销售商品收款**：借 `现金` 30,000；贷 `主营业务收入` 30,000
   同时确认成本：借 `主营业务成本` 15,000；贷 `存货` 15,000
6. **支付工资**：借 `管理费用` 10,000；贷 `现金` 10,000

每笔分录借贷金额相等，保证等式平衡。

## 快速自测

1. 如果企业用自有资金购买办公设备 5 万，会计等式左右分别发生何变化？
2. 赊销商品 2 万，成本 1.2 万，如何写出借贷分录？
3. 为什么说“借方余额”不等于“借了钱”，请解释借贷记账法的逻辑。

### 参考答案与解析

1. 资产：现金减少 5 万，设备增加 5 万 → 资产内部变化，负债与权益不变。
2. 借：`应收账款` 20,000；贷：`主营业务收入` 20,000。成本分录：借 `主营业务成本` 12,000；贷 `存货` 12,000。
3. 借贷记账中的“借/贷”是方向符号，资产增加记借方、负债增加记贷方，与实际资金借入/借出无直接关系。银行账户有借方余额，指的是“资产余额”，并非企业欠银行的钱。
