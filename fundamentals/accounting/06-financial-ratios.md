# 财务比率分析基础 (Financial Ratios)

> 把财务报表数据转化为可比较的指标，快速洞察企业经营状况。

## 学习目标

- 掌握常用的盈利能力、偿债能力、运营效率和成长性比率。
- 理解每个比率的公式、数据来源以及解读方式。
- 能够结合多项比率做出初步诊断，并识别潜在风险。
- 学习比较分析、时间序列分析的基本方法。

## 数据来源说明

- **资产负债表**：资产、负债、所有者权益中的余额（期末或平均值）。
- **利润表**：收入、成本、费用、利润等（通常用年度数据）。
- **现金流量表**：经营现金流等指标。
- **附注信息**：如应收账款周转明细、贷款利率等，有助于细化分析。

## 盈利能力比率

| 指标 | 公式 | 关注点 |
| --- | --- | --- |
| 毛利率 | 毛利 / 营业收入 | 产品定价与成本控制能力 |
| 营业利润率 | 营业利润 / 营业收入 | 主营业务盈利能力 |
| 净利率 | 净利润 / 营业收入 | 整体盈利水平 |
| ROA (资产回报率) | 净利润 / 平均资产 | 资产使用效率 |
| ROE (权益回报率) | 净利润 / 平均所有者权益 | 股东回报水平 |
| 杜邦分析 | ROE = 净利率 × 资产周转率 × 权益乘数 | 拆解 ROE 来源 |

> ROE 高可能来自净利率高（盈利强）、资产周转快（效率高）或杠杆大（权益乘数高）。

## 偿债能力比率

| 指标 | 公式 | 解读 |
| --- | --- | --- |
| 流动比率 | 流动资产 / 流动负债 | 短期偿债能力是否充足 |
| 速动比率 | (流动资产 - 存货) / 流动负债 | 剔除存货后的流动性 |
| 利息倍数 | EBIT / 利息费用 | 支付利息的保障程度 |
| 资产负债率 | 总负债 / 总资产 | 杠杆水平、长期偿债风险 |
| 权益乘数 | 总资产 / 所有者权益 | 财务杠杆强度 |

速动比率高说明流动资产中容易变现的部分充足；利息倍数过低意味着有违约风险。

## 运营效率比率

| 指标 | 公式 | 说明 |
| --- | --- | --- |
| 应收账款周转率 | 营业收入 / 平均应收账款 | 收款速度，结合应收账款周转天数 (365 / 周转率) |
| 存货周转率 | 营业成本 / 平均存货 | 库存管理效率，周转天数 = 365 / 周转率 |
| 总资产周转率 | 营业收入 / 平均总资产 | 资产使用效率 |
| 固定资产周转率 | 营业收入 / 平均固定资产净额 | 固定资产收益性 |

周转率过低可能意味着库存积压、应收账款回收慢；过高则可能意味着库存过低导致断货风险。

## 成长性与现金类指标

- 收入增长率、净利润增长率：同比或复合增长，观察业务扩张速度。
- 自由现金流增长：衡量现金创造能力是否跟上盈利增长。
- 经营现金流/净利润比率：检验利润质量（是否伴随现金流）。

## 分析方法提示

- **横向比较**：与同业平均、行业标杆对比，判断相对强弱。
- **纵向比较**：观察多期趋势，留意增长或下滑趋势。
- **综合判断**：单个比率不能说明全部，需要结合多个指标与定性信息。
- **关注会计策略**：比率受会计估计影响，例如存货计价方法、折旧年限等。

## 实例练习

假设企业数据（单位：千元）：

- 营业收入 500, 营业成本 300 → 毛利 200
- 营业利润 120，净利润 80
- 平均总资产 800，平均所有者权益 400
- 流动资产 350（其中存货 100），流动负债 200
- 应收账款期初 90、期末 110；存货期初 120、期末 80
- 固定资产净额期初 300、期末 260

计算：

1. 毛利率 = 200 / 500 = 40%
2. 净利率 = 80 / 500 = 16%
3. ROA = 80 / 800 = 10%；ROE = 80 / 400 = 20%
4. 流动比率 = 350 / 200 = 1.75；速动比率 = (350-100)/200 = 1.25
5. 应收周转率 = 500 / ((90+110)/2) ≈ 5.0 次；周转天数 ≈ 73 天
6. 存货周转率 = 300 / ((120+80)/2) = 3.0 次；周转天数 ≈ 122 天
7. 固定资产周转率 = 500 / ((300+260)/2) ≈ 1.79 次

> 结合上述指标，可以分析企业盈利能力较好（净利率 16%、ROE 20%），流动性充足，经营效率中等（存货周转天数偏长，需关注库存管理）。

## 快速自测

1. 某企业流动比率为 0.8、速动比率 0.6，这说明什么？若短期债务到期，应采取哪些措施？
2. ROE 大幅上升可能有哪些原因？如何判断这种提升是否可持续？
3. 若经营现金流长期小于净利润，应警惕哪些风险？可以从哪些科目进一步分析？

### 参考答案与解析

1. 流动资产不足以偿还流动负债，短期偿债压力大；需加速回收应收账款、减少库存、争取延长应付款期限或补充流动资金。
2. ROE = 净利率 × 资产周转率 × 权益乘数。原因可能是盈利改善、效率提升或杠杆增加。需分析是否因为一次性收益或过度举债，结合杜邦分解、现金流、利息负担判断可持续性。
3. 可能是应收账款回收慢、存货积压、利润依赖非现金项目等。应检查营运资本科目、非现金费用、资本性支出和债务情况。
