# 常见业务分录 (Journal Entries)

> 把经济业务转换成会计语言：每笔交易都要被准确记录在借方与贷方。

## 学习目标

- 熟悉日常经营活动的典型分录（采购、销售、费用、工资等）。
- 理解调整分录（折旧、摊销、坏账准备、预提费用）的目的。
- 掌握结转与结账流程，知道收入与费用如何结转到本年利润。
- 能够通过简单分录构建三大报表的变化。

## 日常业务分录

| 业务 | 借方 | 贷方 |
| --- | --- | --- |
| 股东投入资本 | 现金 | 实收资本 |
| 银行贷款到账 | 现金 | 短期借款 |
| 购买原材料（现金） | 存货 | 现金 |
| 购买原材料（赊购） | 存货 | 应付账款 |
| 销售商品收现 | 现金 | 主营业务收入 |
| 销售商品赊销 | 应收账款 | 主营业务收入 |
| 确认成本 | 主营业务成本 | 存货 |
| 支付房租 | 管理费用 | 现金 |
| 支付工资 | 管理费用/销售费用 | 现金 |
| 收到客户预付款 | 现金 | 预收账款 |
| 发货确认收入（预收转收入） | 预收账款 | 主营业务收入 |

## 调整分录

会计期末需要调整某些科目以满足权责发生制：

- **折旧**：借 `管理费用/制造费用`，贷 `累计折旧`
- **摊销**（如软件、长期待摊费用）：借 `管理费用`，贷 `累计摊销`
- **坏账准备**：借 `资产减值损失`，贷 `坏账准备`
- **预提费用**（例如已发生但尚未支付的工资、利息）：借 `管理费用`，贷 `应付工资/应付利息`
- **预付费用摊销**（如提前支付保险）：借 `管理费用`，贷 `预付账款`
- **应计收入**（商品或服务已提供，尚未收款）：借 `应收账款`，贷 `主营业务收入`

这些调整确保利润表与资产负债表同时反映真实经营结果。

## 结转与结账

1. **收入结转**：借 `主营业务收入`，贷 `本年利润`
2. **成本与费用结转**：借 `本年利润`，贷 `主营业务成本`、`期间费用`
3. **本年利润结转未分配利润**：若当年盈利，则借 `本年利润`，贷 `利润分配——未分配利润`

结账后，收入/费用类科目余额归零，下一会计期间重新开始累计。

## 简化案例：从分录到报表

假设公司本期业务：

1. 股东投入现金 100,000
2. 购买存货 40,000（现金支付）
3. 赊销商品 60,000，成本 30,000
4. 支付工资 8,000
5. 计提折旧 5,000
6. 期末确认坏账准备 1,000

分录摘要：

- 借 `现金` 100,000；贷 `实收资本` 100,000
- 借 `存货` 40,000；贷 `现金` 40,000
- 借 `应收账款` 60,000；贷 `主营业务收入` 60,000
- 借 `主营业务成本` 30,000；贷 `存货` 30,000
- 借 `管理费用` 8,000；贷 `现金` 8,000
- 借 `制造费用/管理费用` 5,000；贷 `累计折旧` 5,000
- 借 `资产减值损失` 1,000；贷 `坏账准备` 1,000

这些分录将产生以下报表效果：

- **资产负债表**：现金 60,000（100,000-40,000-8,000）、应收账款 59,000（扣除坏账）、存货 10,000（期末余额）、累计折旧 5,000、所有者权益增加等。
- **利润表**：收入 60,000；成本 30,000；管理费用 8,000；折旧 5,000；坏账损失 1,000 → 净利润 16,000。
- **现金流量表**：经营现金流 = 100,000（股东出资） - 40,000（购货） - 8,000（工资） = 52,000；折旧、坏账等为非现金调整。

## 快速自测

1. 预收房租 12,000，此月确认收入 3,000，应如何分录？月末预收账款余额是多少？
2. 当期产生利息费用 4,000 尚未支付，应该做什么分录？对资产负债表有何影响？
3. 为什么要在期末计提坏账准备？如果下期证实应收账款 500 无法收回，应如何处理？

### 参考答案与解析

1. 收到预付款：借 `现金` 12,000；贷 `预收账款` 12,000。确认收入：借 `预收账款` 3,000；贷 `租金收入` 3,000。月末预收余额 9,000。
2. 借 `财务费用` 4,000；贷 `应付利息` 4,000。资产负债表负债增加，应付利息体现未付义务。
3. 坏账准备反映应收账款的风险，提前确认潜在损失。若 500 成为坏账：借 `坏账准备` 500；贷 `应收账款` 500。
