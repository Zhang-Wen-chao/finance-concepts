# 利润表基础 (Income Statement)

> 记录企业在一定期间内的经营成果：赚了多少钱、花了多少钱、最终利润多少。

## 学习目标

- 理解利润表的基本结构：收入、成本、费用、利润。
- 掌握权责发生制如何影响收入、费用的确认。
- 能够从交易构建简化的利润表，并联系资产负债表和现金流量表。
- 识别关键指标，如毛利率、净利率、营业利润等。

## 基本结构

利润表按收入减费用的逻辑排列，常见层次：

1. **营业收入 (Revenue/Sales)**
2. **营业成本 (Cost of Goods Sold, COGS)** → 形成 **毛利 (Gross Profit)**
3. **期间费用**：销售费用、管理费用、研发费用等
4. **营业利润 (Operating Income)**
5. **其他收益/损失**：投资收益、公允价值变动
6. **利润总额 (Profit Before Tax)**
7. **所得税费用 (Income Tax)**
8. **净利润 (Net Income)**

有的企业还会列示“归属于母公司/少数股东”的净利润。

## 权责发生制 vs. 现金制

- 会计采用 **权责发生制**：收入在实现时确认，不必等到收款；费用在发生时确认，不必等到付款。
- 与资产负债表中的应收账款、应付账款等配合，使利润表反映真实经营表现，而非现金流。

## 示例：构建简化利润表

某公司当期数据：

- 营业收入 = 200,000
- 营业成本 = 120,000
- 销售费用 = 20,000
- 管理费用 = 15,000
- 利息费用 = 5,000
- 所得税 = 6,000

利润表：

| 项目 | 金额 |
| --- | --- |
| 营业收入 | 200,000 |
| 营业成本 | (120,000) |
| 毛利 | 80,000 |
| 销售费用 | (20,000) |
| 管理费用 | (15,000) |
| 营业利润 | 45,000 |
| 利息费用 | (5,000) |
| 利润总额 | 40,000 |
| 所得税 | (6,000) |
| 净利润 | 34,000 |

净利润会计入资产负债表的未分配利润，同时影响现金流量表（经营活动现金流）的经营现金。

## 关键指标

- **毛利率** = 毛利 / 营业收入  
  衡量产品定价与成本控制水平。
- **营业利润率** = 营业利润 / 营业收入  
  反映主营业务的盈利能力。
- **净利率** = 净利润 / 营业收入  
  综合体现整体盈利水平。
- **EBIT / EBITDA**：非公认会计准则指标，用于比较企业经营表现。

## 与其他报表的连接

- 利润表的收入、成本会影响资产负债表的应收账款、存货等科目。
- 净利润影响资产负债表的所有者权益（留存收益）。
- 折旧、摊销在利润表中计入费用，但在现金流量表中调整为非现金支出。

## 快速自测

1. 一家公司赊销收入 50,000，成本 30,000，当期只收回 20,000 现金。利润表确认收入和成本多少？资产负债表哪些科目会变化？
2. 净利润为 10,000，但经营活动现金流为 -5,000，可能的原因有哪些？
3. 毛利、营业利润、净利润之间有什么区别？举例说明费用结构对各利润层级的影响。

### 参考答案与解析

1. 利润表确认收入 50,000、成本 30,000；资产负债表应收账款增加 30,000（未收款部分），存货减少 30,000（成本结转）。
2. 可能因为应收账款增加（收入未收现）、存货增加（采购支出）、预付费用、支付以前年度的应付费用等。
3. 毛利 = 收入 - 营业成本，反映产品或服务的直接盈利能力；营业利润再扣除期间费用，查看主营业务盈利；净利润进一步扣除利息、税收、非经常性损益。费用结构不同会导致各层级利润差异，例如研发费用高会压低营业利润但可能改善长期竞争力。
