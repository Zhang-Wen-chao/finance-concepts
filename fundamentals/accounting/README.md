# 会计基础路线图

> 面向零基础学习者，循序渐进掌握企业“财务语言”。

## 学习目标

- 理解会计的核心目的与基本假设。
- 熟悉三大财务报表的结构与相互联系。
- 掌握常见的会计科目分类与记账方法。
- 能够进行基础的财务比率分析，为投资与公司金融学习打底。

## 学习顺序

1. **会计概念入门**
   - 会计的作用、主体与基本假设
   - 会计等式：资产 = 负债 + 所有者权益
   - 借贷记账法的直观理解
2. **三大财务报表**
   - 资产负债表：结构、主要科目、平衡关系
   - 利润表：收入、费用与净利润
   - 现金流量表：经营、投资、筹资活动
   - 报表之间的逻辑链接
3. **日常业务与分录**
   - 采购、销售、费用、薪酬等日常交易的分录
   - 折旧、摊销、坏账准备等调整项目
   - 期末结转与结账流程
4. **财务分析基础**
   - 常见财务比率（盈利能力、偿债能力、运营效率）
   - 杜邦分析框架
   - 简单的趋势分析与同业对标

> 按步骤完成每一部分的笔记与练习，遇到不熟悉的科目及时查阅。

## 学习方式建议

- 结合实际企业财报阅读，加深理解。
- 手写或软件模拟记账分录，掌握借贷思路。
- 关注会计准则变化（如 IFRS 与中国会计准则的差异）。
- 完成基础后，可尝试设计一份简单的财务报表。
