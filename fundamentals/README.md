# 金融基础 (Fundamentals)

金融学的基石，包含经济学、会计学、金融数学和统计学的核心概念。

## 📚 模块内容

### Economics (经济学基础)
- 供给与需求
- 市场机制
- 宏观经济学（GDP、通货膨胀、利率）
- 微观经济学（消费者行为、生产者理论）
- 货币银行学

### Accounting (会计基础)
- 会计基本原理
- 三大财务报表（资产负债表、利润表、现金流量表）
- 财务报表分析
- 财务比率分析

### Financial Mathematics (金融数学)
- 货币时间价值
- 现值与终值
- 年金计算
- 利率与贴现
- 复利与连续复利

### Statistics (统计学基础)
- 概率论基础
- 描述性统计
- 正态分布与其他分布
- 回归分析
- 时间序列分析基础

## 🎯 学习建议

1. **先学经济学基础**：理解市场运作的基本原理
2. **再学会计**：学会读懂企业的"财务语言"
3. **掌握金融数学**：理解金融计算的基础工具
4. **统计学贯穿始终**：为量化分析打基础

这些基础概念是后续学习投资、衍生品、量化金融的前提。
