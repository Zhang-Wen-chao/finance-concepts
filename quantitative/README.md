# 量化金融 (Quantitative Finance)

运用数学、统计学和编程技术进行金融分析和交易。

## 📚 模块内容

### Quantitative Methods (量化方法)
- 时间序列分析
- 随机过程与布朗运动
- 蒙特卡洛模拟
- 回归分析与机器学习
- 因子模型
- 协整与统计套利

### Algorithmic Trading (算法交易)
- 量化策略开发
- 回测框架
- 交易信号生成
- 高频交易 (HFT)
- 执行算法 (TWAP, VWAP)
- 市场微观结构

### Financial Modeling (金融建模)
- Python/R 金融分析
- 数据获取与清洗
- 技术指标构建
- 因子研究
- 风险模型
- 归因分析

### Derivatives Pricing (衍生品定价)
- Black-Scholes 期权定价模型
- 二叉树定价模型
- 蒙特卡洛定价
- 希腊字母 (Greeks)
- 波动率建模
- 利率模型

## 🎯 学习路径

**第一阶段：数学基础**
1. 概率论与统计学
2. 线性代数
3. 随机过程

**第二阶段：编程技能**
1. Python 数据分析 (pandas, numpy)
2. 金融数据获取 (yfinance, tushare)
3. 可视化 (matplotlib, plotly)

**第三阶段：量化策略**
1. 技术分析策略
2. 统计套利
3. 因子投资
4. 机器学习应用

**第四阶段：风险管理**
1. 回测与评估
2. 风险控制
3. 仓位管理

## 💻 常用工具

### Python 库
- **数据处理**: pandas, numpy
- **金融数据**: yfinance, tushare, akshare
- **回测框架**: backtrader, zipline, vectorbt
- **机器学习**: scikit-learn, tensorflow
- **衍生品**: quantlib

### 平台
- QuantConnect
- Quantopian (已关闭，但资源仍有价值)
- Backtrader

## 💡 关键概念

- **数据驱动**：决策基于数据和统计分析
- **回测验证**：策略必须经过历史数据验证
- **风险控制**：量化交易不是圣杯，风控第一
- **过拟合风险**：警惕在历史数据上过度优化
