# Finance Concepts - 金融概念学习指南

> 系统整理金融学的核心概念，用简单易懂的语言解释复杂的金融理论，并记录持续学习的心得

## 🎯 项目简介

这个仓库专注于**金融概念本身**的解释和理解。金融概念是理解市场运作、投资决策和经济规律的基础，掌握了这些概念，你就能在投资实践中做出更明智的决策。

**每个概念都力求：**
- 用简单的语言和类比解释
- 配有适当的例子和图表
- 建立概念间的链接关系
- 避免过度的技术细节，专注概念理解
- 理论与实践结合

## 🎓 推荐学习路径

### 入门路径（适合零基础）
1. **经济学基础** → 理解供需、市场机制、宏观经济
2. **金融市场概览** → 了解股票、债券、衍生品市场
3. **会计基础** → 学会读懂财务报表
4. **投资基础** → 资产配置、风险管理
5. **个人理财** → 将知识应用到实践

### 进阶路径（有基础知识）
1. **投资组合理论** → 现代投资组合理论（MPT）、CAPM
2. **公司金融** → 企业估值、资本结构
3. **固定收益** → 债券定价、久期、凸性
4. **衍生品** → 期权、期货的定价与策略

### 量化路径（结合编程）
1. **金融数学** → 统计学、概率论、随机过程
2. **量化方法** → 因子模型、回归分析
3. **金融建模** → 用Python/R进行金融分析
4. **算法交易** → 量化策略开发与回测

## 💡 学习建议

1. **先建立框架**：先了解金融学的整体结构，再深入细节
2. **理论与实践结合**：学习概念后，看实际案例或真实市场数据
3. **定期复习**：金融概念环环相扣，需要反复巩固
4. **关注实时市场**：理解理论如何在真实市场中运作
5. **量力而行**：从基础开始，不要急于学习高级概念
