# 投资组合理论基础 (Modern Portfolio Theory Basics)

> 现代投资组合理论（MPT）揭示了风险与收益的权衡：借助分散化和优化工具，投资者在既定风险水平下追求最大回报，或在目标收益下最小化风险。

## 学习目标

- 复习风险、收益、相关性等核心概念，并理解分散化的数学直觉。
- 掌握马科维茨均值-方差模型的基本设定、优化目标与约束。
- 认识有效前沿、最小方差组合、切点组合及资本市场线 (CML)。
- 理解 CAPM 的推导逻辑、β、系统性风险与证券市场线 (SML)。
- 了解理论假设、实务应用场景以及主要局限与扩展。

## 核心概念回顾

- **预期收益 (Expected Return)**：投资期间资产收益的加权平均，常用历史均值或主观预测。
- **风险/方差 (Variance)**：收益波动性的度量，标准差 (σ) 体现波动幅度。
- **协方差 (Covariance)** 与 **相关系数 (Correlation, ρ)**：衡量两资产收益同向或反向变化的程度。
- **分散化 (Diversification)**：当资产间相关性低于 1 时，组合整体波动可小于单个资产。
- **风险分解**：总风险 = 系统性风险（无法分散） + 非系统性风险（可通过组合抵消）。

### 二资产组合公式

给定资产 A、B 的权重 `w_A`、`w_B`、收益 `E[R_A]`、`E[R_B]`、方差 `σ_A^2`、`σ_B^2`、相关系数 `ρ`，则组合的预期收益与方差：

```math
E[R_p] = w_A E[R_A] + w_B E[R_B]
```

```math
σ_p^2 = w_A^2 σ_A^2 + w_B^2 σ_B^2 + 2 w_A w_B σ_A σ_B ρ
```

当 `ρ < 1` 时，组合标准差 `σ_p` 可小于权重加权平均，体现分散化收益。

## 马科维茨均值-方差模型

- **目标**：在给定预期收益下最小化组合方差；或在给定风险水平下最大化预期收益。
- **输入**：资产预期收益向量 `μ`、协方差矩阵 `Σ`、权重向量 `w`。
- **优化问题**（最小方差形式）：

```math
\min_{w} \quad w^T Σ w
```

subject to
```math
w^T μ = μ^* \quad ; \quad \mathbf{1}^T w = 1 \quad ; \quad w_i \ge 0 \text{ （若不允许做空）}
```

- **有效前沿 (Efficient Frontier)**：满足收益-风险最优条件的组合集合；曲线左上方不可达，右下方劣于前沿。
- **全局最小方差组合 (GMV)**：在所有可行组合中方差最小的点，收益不一定最高但波动最小。
- **资本市场线 (CML)**：在可投资无风险资产 `R_f` 情况下，最优组合为无风险资产与切点组合 (Tangency Portfolio) 的线性组合，斜率为最大夏普比率。

### 夏普比率 (Sharpe Ratio)

```math
S = \frac{E[R_p] - R_f}{σ_p}
```

衡量每承担一单位标准差风险所获得的超额收益。切点组合是夏普比率最高的风险资产组合。

## CAPM 概览

资本资产定价模型 (CAPM) 将市场组合假设为唯一的系统性风险来源，推导资产预期收益与 β 的线性关系：

```math
E[R_i] = R_f + β_i \left( E[R_m] - R_f \right)
```

- `R_f`：无风险利率；`E[R_m]`：市场组合预期收益。
- `β_i = \frac{\text{Cov}(R_i, R_m)}{σ_m^2}`：资产 i 对市场组合的敏感度。
- **证券市场线 (SML)**：CAPM 的图形表示，横轴 β，纵轴预期收益。
- **阿尔法 (Alpha)**：实际收益与 CAPM 预测收益的偏差，`α = R_i - [R_f + β_i (R_m - R_f)]`。

### CAPM 假设与启示

- 投资者理性、风险厌恶，关注均值-方差；可无成本借贷且投资分割无限可行。
- 资产可无限分割，信息对称；市场完全竞争且无摩擦。
- 推论：所有投资者持有无风险资产与相同的市场组合；超额收益取决于 β。
- 局限：现实存在交易成本、多因子风险、行为偏差；β 可能随时间变化。

## 实务应用

- **资产配置**：估算资产类别的长期均值与协方差，构建有效前沿，选定目标组合。
- **绩效评估**：利用夏普比率、信息比率、α/β 分析基金经理表现。
- **风险预算**：通过边际贡献 (MRC)、成分方差了解各资产对组合风险的贡献。
- **情景分析**：在协方差矩阵不稳定时，使用稳健估计、贝叶斯后验或缩减技术。
- **扩展模型**：多因子模型 (Fama-French)、Black-Litterman、均值方差优化的约束变体、目标风险策略。

## 常见挑战与改进

- **输入敏感性**：均值、协方差估计误差会放大优化结果偏差 → 使用稳健估计、历史窗口、贝叶斯方法。
- **约束处理**：现实中存在仓位限制、税负、交易成本 → 加入线性或非线性约束，采用数值优化。
- **非正态分布与尾部风险**：资产收益可能具备偏度、峰度 → 引入 CVaR、下行风险等替代指标。
- **行为因素**：投资者未必完全理性 → 结合行为金融洞察调整策略。

## 快速自测

1. 为什么当两资产相关系数降低时，组合风险会下降？请用二资产组合公式解释。
2. 什么是有效前沿？切点组合与资本市场线之间有什么关系？
3. CAPM 中 β 的含义是什么？若某资产 β = 1.5、无风险利率 2%、市场超额收益 6%，该资产的理论预期收益是多少？
4. 均值-方差优化对输入数据敏感意味着什么？常见的缓解方法有哪些？

### 参考答案与解析

1. 组合方差含有 `2 w_A w_B σ_A σ_B ρ` 项，相关系数越低，该项贡献越小，整体方差降低；当 ρ < 1 时，组合标准差可能低于单独资产。
2. 有效前沿是所有给定风险下收益最高、或给定收益下风险最低的组合集合；当加入无风险资产时，切点组合与无风险资产连线构成资本市场线，斜率代表最高夏普比率。
3. `E[R_i] = 2% + 1.5 × 6% = 11%`。β 代表资产对市场风险的敏感度，β>1 表示波动大于市场。
4. 输入敏感性指均值、协方差估计误差会导致最优权重剧烈变化。缓解方法包括缩减协方差矩阵、使用贝叶斯/Black-Litterman、设定权重上下限、采用稳健优化或更长历史窗口。
