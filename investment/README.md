# 投资理论与实践 (Investment)

现代投资理论的核心概念，从资产配置到风险管理的完整框架。

## 📚 模块内容

### Portfolio Theory (投资组合理论)
- 现代投资组合理论 (MPT - Modern Portfolio Theory)
- 有效前沿 (Efficient Frontier)
- 马科维茨模型 (Markowitz Model)
- 资本资产定价模型 (CAPM)
- 因子模型 (Factor Models)
- 套利定价理论 (APT)

### Asset Allocation (资产配置)
- 战略资产配置 vs 战术资产配置
- 资产类别（股票、债券、商品、另类投资）
- 再平衡策略
- 目标日期基金
- 风险平价 (Risk Parity)

### Valuation (估值方法)
- 绝对估值法
  - 现金流折现法 (DCF)
  - 股利折现模型 (DDM)
- 相对估值法
  - 市盈率 (P/E Ratio)
  - 市净率 (P/B Ratio)
  - 企业价值倍数 (EV/EBITDA)
- 价值投资 vs 成长投资

### Risk Management (风险管理)
- 风险的类型（市场风险、信用风险、流动性风险）
- 风险度量指标
  - 标准差 (Volatility)
  - VaR (Value at Risk)
  - 夏普比率 (Sharpe Ratio)
  - 最大回撤 (Maximum Drawdown)
- 对冲策略
- 分散化投资

## 🎯 学习路径

1. **投资组合理论** → 理解风险与收益的权衡
2. **资产配置** → 学会构建投资组合
3. **估值方法** → 判断资产是否值得投资
4. **风险管理** → 保护你的投资组合

## 💡 关键概念

- **分散化**：不要把鸡蛋放在一个篮子里
- **风险收益权衡**：高收益通常伴随高风险
- **长期投资**：时间是投资者的朋友
- **理性决策**：避免情绪化交易
